# homebridge-horizonbox
Homebridge Plugin to toggle Power of Horizon Box

## Usage

config.json sample:

```json
"accessories": [
   {
       	"accessory": "horizon",
       	"name": "Box",
       	"ip": "192.168.178.62"
   }
]
```


Default Values for config.json:
name: 		"Led-Strip"
brightness:	false
